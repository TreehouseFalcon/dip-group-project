CSCE 5225 Digital Image Processing

Project

 

The project is a group assignment. It is evaluated based on the source codes, report, and presentation. The source code and report must be submitted to the Group project (code and report) portal and the slides and presentation must be submitted to the Project Slides and Presentation portal in Canvas. The programming language for the project is MATLAB. If multiple files are to be submitted, you may combine them using zip compression. The group is evaluated as a whole but a weight is assigned to each member to calculate the individual grade.

 

Each student will have a chance to evaluate his/her team members, including him/herself in this evaluation. The average evaluation from all members will be used as the weight to decide the final project grade of each student. This evaluation is anonymous to all the students and will be conducted via a quiz.

 

All projects will be evaluated for the correctness and readability of the Graphical User Interface (GUI), the code, the report, and the presentation. The report needs to document how to run/use the program and give demonstrative cases to show the functionalities. The presentation will walk through the implementation of the functionalities and give live demonstrations of the programs.

 

The total points for the project is 400.

·         For the code part, completeness, correctness, and readability are the evaluation aspects. Up to 10 points can be deducted for missing meaningful comments and proper structuring of the source code.

·         For the report, a description of various functions available in the GUI and how to use it takes 20 points and the rest 10 points are for the demonstrative cases.

·         The slides are evaluated based on the organization (10 points), readability (5 points), and coverage (15 points).

·         The presentation is evaluated based on the clearness of explanation (40 points) and live demonstration (20 points).
